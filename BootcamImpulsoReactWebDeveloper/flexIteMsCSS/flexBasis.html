Estabelece o tamanho incial do item antes das distribuições de espaço restante dentro dele, usando como base o conteúdo interno disposto.

Valores possíveis:
- auto: se o item não ter tamanho, ele será porporcional ao conteúdo do item;
- px, &, em,...: são valores exatos;
- 0(zero): relação com a definição do flex-grow.

Exemplos:

auto

A
B
C
DEF

0 (zero)

AAAA
B
C
DEF

basis + width

AAA
BBB
CCC
DEF
